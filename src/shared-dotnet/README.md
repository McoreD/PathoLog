# Shared .NET Libraries

Shared C# libraries for `apps/web-csharp` and `apps/wpf`.

## Projects
- `PathoLog.Domain`: entities, enums, and value objects.
- `PathoLog.Contracts`: DTOs and strict JSON schemas for AI extraction.
- `PathoLog.Extraction`: PDF text extraction and OCR interfaces.
- `PathoLog.Mapping`: analyte_short_code generation and learning logic.
- `PathoLog.Persistence`: local database access and migrations.
- `PathoLog.Trending`: trend queries and calculations.
